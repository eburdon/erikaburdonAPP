About This Site

This is my full stack experimental playground! For the front end, I've tried to focus on good design and awesome UI, keeping mobile layouts a priority. It's built with Angular! It also uses Sass and Foundation for Apps. I'm focusing on following good coding practices, particularly flexible and easy-to-maintain components, as well as clear code and good performance.

The back end's primary language is Scala, but I'm tinkering with other languages such as Haskell, and trying to use Amazon Lambda and other neat-o things sometime soon. But first, I plan to build an Android app that will make real changes to this site - that project will finish my Front End, Back End, and Mobile triangle.
