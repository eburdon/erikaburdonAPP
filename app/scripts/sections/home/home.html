Distributed Facebook Analysis

Creating open-source scripts that can analyze a user's Facebook data. Only users can download their information and execute this script. The FB data set includes information such as photo and message metadata from the day they opened an account in an easy-to-parse XML format. The first iteration of this project is to analyze their message history. This is written in Go, and distributes the workload via a MapReduce implementation. Data is stored and accessed using an efficient Prefix Trie. The final output is a pie chart of their most active hours. Future work includes further partitioning and distribution of the workload, migrating the Python parser to Go, and further message analysis such as most active day of the week, month, year, etc.
