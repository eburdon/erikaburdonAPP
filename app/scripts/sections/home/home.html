About This Site

This is my full stack experimental playground! For the front end, I've tried to focus on good design and awesome UI. I'm experimenting with different styles on each page - I'm looking to see what does and doesn't work as a user, and what's easy to implement and maintain. I hope to include some style guides on the code base as well.

The back end's primary language is Scala, but I'm tinkering with other languages such as Haskell, using Amazon Lambda, and other neat-o things sometime soon. Once the basic pages of this site are done, I intend to build a simple Android app that uses the same endpoints as this site to make real changes on the front end.
