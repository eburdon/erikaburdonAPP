Angular Web App

Research and Education Platform

GitHub Tutorial

For a research course, teams were required to create some automatic tool or script that would investigate a code base to see how they evolved over time, and present a textual description and visual result of the tool.
My partner and I decided to build a single-page Angular web app to unify the entire class's research into one cohesive tool. Visitors are able to both execute any of the 10 tools, then view the freshly-generated visual results within the app. Links and summaries to the original GitHub repositories were also included.

Python GUI and Scripting Program

Word-of-the-day Email Parser

GitHub Tutorial

In an effort to gain more exposure to German, I subscribed to a mailing list which would send up to 2 emails per day of a new vocabulary word. These included the word, its translation, and example sentence in both German and English.
I wrote a script to parse each into a text file to study later, then delete the email. Result included a Python GUI to enter log-in information, then select either POP3 or IMAP access.
